# Myra UI Generator Wiki

Welcome to the Myra UI Generator documentation wiki!

## What is Myra UI Generator?

Myra UI Generator is a source generator for Myra UI XML files that automatically creates strongly-typed C# accessor classes. It scans your Myra XML files at compile-time and generates C# classes with properties for each widget, providing type-safe access with full IntelliSense support.
